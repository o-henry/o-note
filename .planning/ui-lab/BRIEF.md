# UI Rebuild Brief

## Goal

Rebuild the o-note shell so it feels like Bullpen: dense, local, precise, and operational.

## Must Keep

- Markdown and HTML note workflows.
- Split, preview, source, export, annotation, and compare modes.
- Search and keyboard navigation.
- Import, export, backup, integrity check, and search repair controls.
- Existing test-visible labels and ARIA roles.

## Visual Direction

- 17rem sidebar, content inset, status strip, report header, underline tabs, main document surface, right status rail.
- Zero-radius controls and panels.
- Hairline borders, no shadows.
- Mono uppercase labels, counts, statuses, and compact meta.
- Dark neutral Bullpen-style palette with paper-like document previews.
